# Student CGPA Predictor

## Project Overview
This project predicts a student's CGPA based on:
-Hours Studied
-Attendance Percentage

## Technologies Used
-Python
-Pandas
-Scikit-Learn
-Joblib

## Features
-Read student data from CSV
-Train a Linear Regression model
-Save trained model
Predict CGPA for new students
